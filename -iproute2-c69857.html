交叉编译 iproute2

Posted on Mon, Mar 4, 2024 linux embedded cross_compile

下载iproute2 源码, 此处选择 版本号为5.9.0 的源码

tar -xvf iproute2-5.9.0.tar.xz
cd iproute2-5.9.0 && mkdir output
./configure --prefix=/home/sirius/cross_compile/iproute2-5.9.0/output --host=aarch64-none-linux-gnu

修改 Makefile 以及 config.mk 文件中的交叉编译器名称

例如config.mk 中

AR:=ar
CC:=gcc

修改为

AR:=aarch64-none-linux-gnu-ar
CC:=aarch64-none-linux-gnu-gcc

Makefile 中SUBDIRS 仅保留 lib 和 ip

然后执行make && make install
