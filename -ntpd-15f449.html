交叉编译 ntpd

Posted on Tue, Jan 16, 2024 linux embedded cross_compile

首先编译 openssl1.1.1w

./Configure --prefix=${PWD}/output --openssldir=${PWD} linux-aarch64 CC=aarch64-none-linux-gnu-gcc
make
make install

然后编译 ntpd

./configure --prefix=${PWD}/output --host=aarch64-none-linux-gnu CC=aarch64-none-linux-gnu-gcc --with-yielding-select=yes --with-openssl-incdir=${PWD}/../openssl-1.1.1w/output/include --with-openssl-libdir=${PWD}/../openssl-1.1.1w/output/lib --with-ntpsnmpd=no
make 
make install
