交叉编译 psmisc

Posted on Wed, Aug 21, 2024 cross_compile linux

首先编译 ncurses

编译完成后, 记下ncurses 编译生成的include 和 lib路径分别添加到 CFLAGS 以及 LDFLAGS环境变量中

下载 psmisc 源码

wget https://gitlab.com/psmisc/psmisc/-/archive/v23.7/psmisc-v23.7.tar.gz
tar -xvf psmisc-v23.7.tar.gz 
cd psmisc-v23.7 && ./autogen.sh
mkdir -p output
export CFLAGS += {}
./configure --prefix={PWD}/output --host=aarch64-none-linux-gnu --libdir={ncurses lib 路径} --includedir={ncurses include 路径}
make

此时会遇到找不到curses.h 以及 term.h 的报错, 将缺少的头文件拷贝至psmisc/src/ 下即可

最后 make install
