交叉编译 htop

Posted on Tue, Dec 26, 2023 linux embedded cross_compile

首先需要编译 ncurses

git clone https://github.com/gittup/ncurses
cd ncurses
./configure --target=aarch64-none-linux-gnu --with-gcc=aarch64-none-linux-gnu-gcc --with-shared --host=aarch64-none-linux-gnu --with-build-cpp=aarch64-none-linux-gnu-g++ --disable-unicode
make

然后编译 htop

git clone https://github.com/htop-dev/htop
./autogen.sh
export CPPFLAGS=-I/tmp/ncurses/include
export LDFLAGS=-L/tmp/ncurses/lib
./configure --target=aarch64-none-linux-gnu --with-gcc=aarch64-none-linux-gnu-gcc --with-shared --host=aarch64-none-linux-gnu --with-build-cpp=aarch64-none-linux-gnu-g++ --disable-unicode
make
