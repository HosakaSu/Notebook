Powershell 设置代理

Posted on Tue, Dec 26, 2023 Windows GFW

查看代理

netsh winhttp show proxy

设置代理 (需要管理员权限)

netsh winhttp set proxy 127.0.0.1:7890

或者不需要代理的设置方式

$env:HTTP_PROXY="http://127.0.0.1:7890"
$env:HTTPS_PROXY="https://127.0.0.1:7890"

取消代理

netsh winhttp reset proxy
