交叉编译Python

Posted on Thu, Jan 18, 2024 linux embedded cross_compile

以python 3.8.9 为例, 下载python 源码

首先要编译主机端Python

./configure --prefix=${PWD}/host_pc_output --enable-optimizations
make && make install

然后开始编译设备端python

首先交叉编译 zlib

export CC=aarch64-none-linux-gnu-gcc
export CROSS_PREFIX=aarch64-none-linux-gnu-
export CFLAGS="-Wall -g"
./configure --prefix=${PWD}/output 
make && make install

交叉编译 libffi

./configure --prefix=${PWD}/output  --host=aarch64-none-linux-gnu --target=aarch64-none-linux-gnu CC=aarch64-none-linux-gnu-gcc CXX=aarch64-none-linux-gnu-g++ RANLIB=aarch64-none-linux-gnu-ranlib STRIP=aarch64-none-linux-gnu-strip AR=aarch64-none-linux-gnu-ar CFLAGS="-Wall -g" CPPFLAGS="-Wall -g"
make && make install

交叉编译 libressl

./configure --prefix=${PWD}/output --host=aarch64-none-linux-gnu
make && make install

然后开始交叉编译Python

./configure --prefix=\$\{PWD\}/device_output --host=$CROSS_COMPILE --build=x86_64-pc-linux-gnu --disable-ipv6 CC=$CROSS_COMPILE-gcc CXX=$CROSS_COMPILE-g++ RANLIB=$CROSS_COMPILE-ranlib STRIP=$CROSS_COMPILE-strip AR=$CROSS_COMPILE-ar LDFLAGS="-L/home/sirius/cross_compile/zlib-1.2.13/output/lib -L/home/sirius/cross_compile/libffi-3.4.4/output/lib64" LIBS="-lz -lffi" CFLAGS="-Wall -g -I/home/sirius/cross_compile/zlib-1.2.13/output/include -I/home/sirius/cross_compile/libffi-3.4.4/output/include" CPPFLAGS="-Wall -g -I/home/sirius/cross_compile/zlib-1.2.13/output/include -I/home/sirius/cross_compile/libffi-3.4.4/output/include" ac_cv_file__dev_ptmx=no ac_cv_file__dev_ptc=no --enable-optimizations --with-openssl=/home/sirius/cross_compile/libressl-3.8.2/output

安装pip

wget https://bootstrap.pypa.io/get-pip.py

安装pip

python3 get-pip.py

如果有代理需要, —proxy=http://ip:port 即可
